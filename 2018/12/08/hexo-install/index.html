Hexo安装与配置

安装hexo

Hexo是node的一个前端静态资源框架，因此需要先安装node.js和npm，此处假设已经安装好node相关内容。

初始化一个目录为hexo主目录，然后安装hexo

More info: Setup

配置hexo的主题

hexo的主题，位于<floder>/themes目录中，进入目录，然后下载对应的主题，此处以yilia为例

打开hexo的配置文件_config.yml，修改theme字段的值为下载的主题：yilia.
